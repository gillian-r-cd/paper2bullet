## 第一层共识：「命名困境」作为实操抓手

### 命题

对每一个 aha 候选，回答一个问题：

> **它命名了谁的什么困境？**
> 

> 格式：「___（角色）一直在经历___（现象），因为他们不知道___（这个发现揭示的结构）」
> 

如果这句话填不出来，或者填出来后觉得"这个困境没那么普遍"，大概率不合格。

### 为什么这个测试有效

它同时锁死了多个看似独立的约束：

- **受众**：你必须知道他们正在经历什么，否则无法指认困境
- **主题**：困境必须落在课程领域内，否则命名了也没用
- **课程叙事**：一个被命名的困境天然有叙事张力
- **实用性**：困境一旦被命名，行动方向自动出现
- **商业价值**：人为"终于有人说清楚了我一直在挣扎的事"买单

### 归纳来源

这个命题来自对正负样本判据的语言模式分析——不看论文内容本身，而看**老板的判据中反复出现的结构**：

**正样本判据的共同结构**：每一条都能指向"某类人正在被某件事卡住"

- P2（任务难度与努力）："体感上，老销售都不愿意卖新产品"
- P3（期望-失望理论）："功能性的角色期望都能训练"
- P4（目的论框架）："三种策略对同一个问题进行思考"
- P5（AI元认知支架）："论文结论可以直接用在演讲中"

**负样本判据的共同结构**：每一条都不能指向具体的人在经历具体的什么

- N1（技术可供性）："对当前课程体系帮助比较间接"
- N2（JD分析）："分析JD的意义一般"
- N3（理论框架汇编）："理论框架梳理有帮助，但无法直接运用"

## 第二层共识：「因果重构」作为底层必要条件

### 推演路径

「命名困境」有效，但它是代理指标（proxy），不是本体。追问"为什么命名困境就能锁死那些约束"，触及到更深的一层：

> **Aha moment 的本体是：受众对某件已有经验之事的因果解释被重构。**
> 

三个组成部分：

- **已有经验**：受众必须亲身经历过这件事。没经历过，就没有可被重构的旧模型，aha 无处发生
- **因果解释**：受众对这件事有一个现成的因果归因（哪怕是错的、模糊的、隐性的）
- **被重构**：论文提供了一个不同的因果结构，替换或重组了旧归因

### 验证

| 正样本 | 旧因果模型 | 新因果模型 |
| --- | --- | --- |
| **P1** 语言风格理论 | 说服力是一种天赋或语感 | 说服力由六种可检测的语言标记决定 |
| **P2** 任务难度与努力 | 老销售不卖新品是因为懒或抗拒变化 | 产品优势太明显时，任务太简单反而抑制努力 |
| **P3** 期望-失望理论 | 客户满意取决于产品好坏 | 满意度取决于期望与感知表现之间的差值结构 |
| **P4** 目的论框架 | 做决策就是分析利弊然后选 | 存在三种时间取向，各自生成完全不同的决策逻辑 |
| **P5** AI元认知支架 | AI教学可能更好但说不清为什么 | AI产生更高的生成性认知负荷和反思循环，有定量证据 |

| 负样本 | 为什么不构成因果重构 |
| --- | --- |
| **N1** 技术可供性 | 提供了分类语言，但没有替换任何已有的因果归因 |
| **N2** JD分析 | 描述了职位分布现状，是信息增量，不改变因果归因 |
| **N3** 六个理论框架 | 汇编了已有因果模型，但汇编本身不构成重构——知道更多理论 ≠ 因果解释被改变 |
| **N4** 旅行社数据 | 有潜力重构（封闭式 > 开放式问题），但1982年单一样本意味着新因果模型不可信——重构要成立，新解释的证据强度必须高于旧解释 |
| **N5** GenAI生产力分类 | 分类框架不改变因果结构，且内容在2026年对目标受众已是常识 |

### 关键区分：信息增量 vs. 因果重构

这一层共识揭示了一条判断规则的底层逻辑：**分类 ≠ 洞见**。分类（N1、N2、N5）增加了描述维度，但没有替换因果结构，所以永远不产生 aha。汇编（N3）堆叠了多个已有模型，但堆叠本身不改变受众对任何一件事的因果解释。

### 「命名困境」与「因果重构」的关系

「命名困境」是「因果重构」的高效实操代理。当你能命名困境时，你已经隐式地完成了三件事：确认受众有这个经验、确认他们现有的解释不够用、确认论文提供了新的解释。困境是入口，重构是 aha 本身。

## 第三层共识：完整的四条件模型

### 推演路径

在确立因果重构为 aha 本体之后，对正负样本的进一步审查揭示了一个独立维度：**新因果模型必须具备被转化为课程内容的潜力**。

这意味着 aha moment 承担着双重身份：

- 作为**认知事件**：它重构了受众的因果解释
- 作为**教学对象**：它必须能被转化为方法论、佐证、框架、叙事等课程产物

这两个身份是独立变量——一个发现可以深刻地重构因果解释，但新模型的变量是抽象的、不可操作的，在课程里找不到落点；反过来，一个发现的重构深度一般，但新模型直接交付了一个可立刻使用的操作框架。

### 四个必要条件（缺一致命）

<aside>
🔑

**1. 因果重构**：受众对某件已有经验之事的因果解释被重构

**2. 方法论可信**：新因果模型有充分的证据支撑

**3. 认知活跃性**：旧因果模型在目标受众的当下认知中仍然活跃

**4. 课程可操作化**：新因果模型可以被转化为课程中的可落地内容——把一个之前模糊或错误的解法更新成清晰且可落地的解法，把未揭示或错误的命题转化成已揭示或更有说服力的命题

</aside>

各条件被违反时的失败模式：

| 违反的条件 | 失败模式 | 样本例证 |
| --- | --- | --- |
| 条件1 缺失 | 信息增量，非因果重构。分类、汇编、现状描述。学员听完说"哦知道了"但没有"原来如此" | N1 技术可供性、N2 JD分析、N3 理论汇编、N5 GenAI分类 |
| 条件2 缺失 | 新因果模型站不住脚。课堂上第一个被挑战的就是"这个数据靠谱吗"，瓦解 aha 的力量 | N4 旅行社数据（1982年单一样本） |
| 条件3 缺失 | 重构了一个已经没人持有的旧信念。对2026年的目标受众而言是 low-hanging fruit，"AI能帮你写代码"不再是 aha | N5 GenAI生产力（2023-2024要点缺乏时代代表性） |
| 条件4 缺失 | 因果重构发生了，但新模型是抽象的、不可观测的、不可操作的——学员理解了但用不上，课程没有产物 | N1 技术可供性（即使接受了affordance分类，课程中无法落地为方法或练习） |

## 推演：模型的操作性后果

### 推演一：条件之间存在部分补偿关系

P1（语言风格理论）在条件1（因果重构深度）上得分低——它不深刻也不反直觉，但在条件4（课程可操作化）上得分极高——六种弱力语言标记直接变成评价 rubric。它过关是因为条件4的强度弥补了条件1的阈值。

这解释了原文档中的隐性规则：**"可操作性可以弥补反直觉性的不足"**。

但补偿有下限。如果因果重构完全不存在——仅仅提供了一个操作清单但没有改变任何理解——那就是培训手册。

### 推演二：条件4有内部结构——四种操作化模态

课程可操作化至少有四种模态，它们在课程中承担不同的功能：

| 操作化模态 | 课程中的功能 | 对应正样本 |
| --- | --- | --- |
| **方法/工具** | 学员可以直接拿去用的操作程序 | P1（语言标记评价维度）、P4（三种决策姿态作为思维工具） |
| **佐证/论据** | 支撑课程中某个观点的实证数据 | P5（AI教学优于传统教学的定量证据） |
| **重构性叙事** | 改变学员对某个场景的理解方式 | P2（新产品销售阻力的反直觉归因） |
| **框架/模型** | 给学员一个新的分析镜头 | P3（期望-失望模型） |

同一个 aha moment 可能适用多种模态：P4 既可以是"方法/工具"（让学员用三种姿态分析同一个问题），也可以是"框架/模型"（让学员理解为什么不同人的决策逻辑看起来不可调和）。

课程编排需要不同模态的交替来维持节奏，因此产出菜单应按模态分类。

### 推演三：「多重制约感」来自两个独立评估被混在一起做了

最初的困惑——"要同时满足受众、主题、叙事、实用性，很难下判断"——来自于在一次判断中**同时运行两个本应串行的评估**：

- **阶段 A**（这是 aha 吗？）：条件 1 + 2 + 3。评估认知事件本身的质量
- **阶段 B**（这个 aha 能用吗？）：条件 4。评估新模型的操作化潜力

串行执行的好处：A 失败的东西不需要进入 B——省掉了大量认知负荷。B 失败的东西可以暂存，因为课程设计变化后可能重新获得操作化路径。

# 筛选过程

### 核心逻辑：每个阶段只能用当前可获得的信息来评估四条件的子集

四条件分别是：①因果重构 ②方法论可信 ③认知活跃性 ④课程可操作化。在论文处理的不同阶段，你手上拥有的信息量不同，因此能检验的条件也不同。筛选漏斗的结构应该从这个事实推导出来，而不是凭直觉设计。

### 阶段 0：论文发现（目标是覆盖率，不做过滤）

这一步的任务是生成候选池。你的文档里已经定义了两类来源——超级经典 + 最前沿实证。这是**搜索策略**，不是筛选标准。这一步唯一的约束是**领域边界**：论文必须与课程的主题域有关联。但这个关联可以是宽泛的，因为有些跨领域论文恰恰是最好的aha来源（比如P4的目的论框架来自复杂性科学）。

这一步不需要做任何四条件的评估。

### 阶段 1：元数据筛选（标题 + 摘要 + 发表信息）

**你拥有的信息**：标题、摘要、作者、发表年份、期刊/会议、引用量、关键词。

**你能评估什么**：

条件②（方法论可信）的**粗筛代理**：

- 发表渠道的可信度（顶刊 vs. 掠夺性期刊）
- 摘要中描述的方法论类型（实验 / 准实验 / 调查 / 案例研究 / 纯理论推演）
- 样本量级别（如果摘要提及）

条件③（认知活跃性）的**粗筛代理**：

- 论文的发现在2026年是否已经是常识？这在摘要层面有时能直接判断（"AI can assist in customer service" 级别的结论一看就过时了）

**你能做出的最关键的结构性判断**——论文类型排除：

- 摘要是否表明这是一篇**纯分类/纯综述/纯理论盘点**性质的论文？如果是，它在结构上就不可能产出因果重构——因为它的目标是汇编已有知识，而汇编本身不重构任何人的因果解释（这就是N3的失败模式）
- 注意：一篇综述如果**在综述过程中提出了新的整合性框架**，那它的综述部分不过关，但那个新框架可能过关。所以这里的排除应该标注为"低优先级"而非"直接丢弃"

**你不能评估什么**：

- 条件①（因果重构）：摘要通常只告诉你"我们发现了X"，但你无法判断X是否重构了受众的因果模型——因为你还没看到X的具体内容和论证
- 条件④（课程可操作化）：你完全不知道发现的落地形态

**阶段 1 的决策规则**：

直接排除（任一条满足）：

- 明确的领域不相关
- 摘要结论对2026年目标受众已是显而易见的常识
- 方法论有硬伤信号（样本量极小 + 极旧 + 单一来源，且摘要中没有方法论创新的补偿）

低优先级（降序处理，不丢弃）：

- 纯综述/纯分类论文（结构上不利于因果重构，但可能内含新框架）
- 摘要结论含糊、无法判断是否有具体发现

通过（进入阶段 2）：

- 其余所有论文。**宁多勿漏**——在这个阶段，假阴性的代价远高于假阳性

### 阶段 2：快速阅读（引言 + 结果 + 讨论 + 关键图表）

**你新增的信息**：论文的核心发现、论证结构、数据质量、关键图表。

**你现在能评估什么**：

条件①（因果重构）**首次可评估**：

- 论文的发现是否提供了一个**新的因果解释**，替代了受众可能持有的旧解释？
- 用「命名困境」测试快速判断：能否填出「___一直在经历___，因为他们不知道___」？
- 如果论文通篇在做描述（"我们发现了X的分布情况"）、分类（"我们把Y分为三类"）、或重复验证（"我们复制了前人的发现Z"），那它大概率不构成因果重构

条件②（方法论可信）**精确评估**：

- 样本量、采样方式、数据年代、分析方法是否支撑其结论？
- 这篇论文的方法论弱点，是否会在课堂上成为第一个被质疑的点？

条件③（认知活跃性）**更精确评估**：

- 现在你看到了具体发现，可以更准确地判断：目标受众是否仍然持有被挑战的旧信念？

条件④（课程可操作化）**首次可粗筛**：

- 你开始有一个直觉：这个发现能变成什么课程产物？但此时不需要精确回答——只需要判断"有没有可能性"

**阶段 2 的决策规则**：

排除：

- 条件①明确不满足（无因果重构，纯描述/纯分类/纯验证）
- 条件②明确不满足（方法论硬伤，N4模式）
- 条件③明确不满足（结论对目标受众已是常识）

进入卡片提取（阶段 3）：

- 条件①②③均未被否决，且至少有一个发现看起来有因果重构的潜力

### 阶段 3：卡片提取（完整的四条件评估）

这就是你现有工作流中已经定义好的部分：逐张卡片做完整的四条件判断，标三色，输出结构化卡片。

### 整体漏斗的设计原则

有一个值得注意的推论：**阶段1和阶段2的筛选标准都是"否决式"的**——它们只回答"这篇论文是否*肯定不行*"，而不试图回答"这篇论文是否*肯定行*"。这跟你的三色标记逻辑是一致的：绿色是阶段3才能给的判断，前两个阶段只能给红色和"通过到下一阶段"。
